# Первый выезд

Цель первого выезда — **понять процесс**, а не собрать “идеальный арсенал”.

## Перед выходом (5 минут)
- Проверь **правила региона**: где можно ловить, нормы вылова, запреты на наживки/крючки.
- Возьми/оформи **разрешение/лицензию** (если требуется).

## Минимальный набор (берег, пресная вода)
- Удилище + катушка
- Леска (для старта часто выбирают моно)
- Набор грузил
- Крючки (малые размеры под поплавок/насадки)
- Поплавок
- Наживка или простые приманки (и обязательно проверь, что это разрешено правилами)
- Подсачек (желательно резиновый/мягкий)
- Перчатка/защита для рук (опционально)

## Инструменты, которые реально спасают
- Плоскогубцы/экстрактор (снимать крючок)
- Кусачки/ножницы для лески
- Мини-аптечка (пластырь, антисептик)
- Пакет для мусора/лески (забирай всё с собой)

## По погоде
**Солнце / жара**
- кепка/панама, крем SPF, вода

**Ветер / дождь**
- непромокаемая куртка, запасные перчатки, чехол для телефона

**Холод**
- слои одежды, тёплые носки, термос

## По типу водоёма
**Трава / коряги**
- больше зацепов → бери запас крючков/грузил

**Камни / ракушка**
- выше истирание → подумай о поводке/более стойкой леске

**Мутная вода**
- часто важнее заметность приманки и проводка, чем “невидимая” леска

## Эко-правило для новичка
Оставленная леска и мусор опасны для животных. Всё, что принёс — унеси.
